## 快速开始

### 1. 安装 Neovim v0.11+
```bash
brew install neovim
```

### 2. 克隆本仓库
```bash
git clone https://github.com/Dwsy/nvim-config ~/.config/nvim
```

### 3. 启动并安装
```bash
nvim
:MasonInstallAll    # 安装所有 LSP / 工具
```

### 4. 测试 LSP
```bash
nv your-project/    # 打开项目
gd                  # 试试跳转定义
```
