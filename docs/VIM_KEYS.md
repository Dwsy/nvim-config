# Neovim 基础快捷键

## 模式切换

| 快捷键 | 功能 |
|--------|------|
| `i` | 进入插入模式 (光标前) |
| `a` | 进入插入模式 (光标后) |
| `I` | 行首插入 |
| `A` | 行尾插入 |
| `o` | 下方新建行并插入 |
| `O` | 上方新建行并插入 |
| `v` | 可视模式 (字符选择) |
| `V` | 可视行模式 (整行选择) |
| `Ctrl + v` | 可视块模式 (列选择) |
| `Esc` / `Ctrl + [` | 返回普通模式 |
| `:` | 命令行模式 |
| `R` | 替换模式 |

## 光标移动

### 基础移动

| 快捷键 | 功能 |
|--------|------|
| `h` | 左 |
| `j` | 下 |
| `k` | 上 |
| `l` | 右 |
| `w` | 下一个单词开头 |
| `W` | 下一个单词开头 (空格分隔) |
| `b` | 上一个单词开头 |
| `B` | 上一个单词开头 (空格分隔) |
| `e` | 当前/下一个单词结尾 |
| `E` | 当前/下一个单词结尾 (空格分隔) |

### 行内移动

| 快捷键 | 功能 |
|--------|------|
| `0` | 行首 |
| `^` | 行首非空字符 |
| `$` | 行尾 |
| `f{char}` | 跳到本行下一个 {char} |
| `F{char}` | 跳到本行上一个 {char} |
| `t{char}` | 跳到 {char} 前一个字符 |
| `T{char}` | 跳到 {char} 后一个字符 |
| `;` | 重复上次 f/F/t/T |
| `,` | 反向重复 f/F/t/T |

### 文件内移动

| 快捷键 | 功能 |
|--------|------|
| `gg` | 文件开头 |
| `G` | 文件末尾 |
| `{n}G` / `:{n}` | 跳到第 n 行 |
| `Ctrl + f` | 下翻一页 |
| `Ctrl + b` | 上翻一页 |
| `Ctrl + d` | 下翻半页 |
| `Ctrl + u` | 上翻半页 |
| `H` | 屏幕顶部 |
| `M` | 屏幕中间 |
| `L` | 屏幕底部 |
| `zz` | 当前行居中 |
| `zt` | 当前行置顶 |
| `zb` | 当前行置底 |
| `%` | 跳转到匹配的括号 |

### 跳转

| 快捷键 | 功能 |
|--------|------|
| `Ctrl + o` | 跳回上一个位置 |
| `Ctrl + i` | 跳到下一个位置 |
| `''` | 跳回上次编辑位置 |

## 编辑操作

## 搜索替换

### 搜索

| 快捷键 | 功能 |
|--------|------|
| `/{pattern}` | 向下搜索 |
| `?{pattern}` | 向上搜索 |
| `n` | 下一个匹配 |
| `N` | 上一个匹配 |
| `*` | 搜索光标下单词 (向下) |
| `#` | 搜索光标下单词 (向上) |
| `:noh` | 取消高亮 |

### 替换

| 命令 | 功能 |
|------|------|
| `:s/old/new` | 替换当前行第一个 |
| `:s/old/new/g` | 替换当前行所有 |
| `:%s/old/new/g` | 替换全文所有 |
| `:%s/old/new/gc` | 替换全文 (逐个确认) |
| `:5,10s/old/new/g` | 替换 5-10 行 |

## 文本对象

配合 `d` `c` `y` `v` 使用:

| 对象 | 说明 |
|------|------|
| `iw` | 内部单词 |
| `aw` | 单词 (含空格) |
| `is` | 内部句子 |
| `as` | 句子 (含空格) |
| `ip` | 内部段落 |
| `ap` | 段落 (含空行) |
| `i"` | "" 内部 |
| `a"` | "" 包含引号 |
| `i'` | '' 内部 |
| `i)` / `ib` | () 内部 |
| `a)` / `ab` | () 包含括号 |
| `i]` | [] 内部 |
| `i}` / `iB` | {} 内部 |
| `it` | tag 内部 |
| `at` | tag 包含标签 |

**示例:**
- `diw` - 删除单词
- `ci"` - 修改引号内内容
- `ya}` - 复制 {} 及内容
- `vip` - 选中段落

## 标记 (Marks)

| 快捷键 | 功能 |
|--------|------|
| `m{a-z}` | 设置标记 (文件内) |
| `m{A-Z}` | 设置全局标记 |
| `'{mark}` | 跳到标记行 |
| `` `{mark} `` | 跳到标记位置 |
| `:marks` | 查看所有标记 |
| `:delmarks a` | 删除标记 a |

## 宏录制

| 快捷键 | 功能 |
|--------|------|
| `q{a-z}` | 开始录制宏到寄存器 |
| `q` | 停止录制 |
| `@{a-z}` | 执行宏 |
| `@@` | 重复上次宏 |
| `5@a` | 执行宏 5 次 |

## 文件操作

| 命令 | 功能 |
|------|------|
| `:w` | 保存 |
| `:w {file}` | 另存为 |
| `:q` | 退出 |
| `:q!` | 强制退出 |
| `:wq` / `:x` / `ZZ` | 保存并退出 |
| `:e {file}` | 打开文件 |
| `:e!` | 重新加载文件 |
| `:bn` / `:bp` | 下/上一个 buffer |
| `:bd` | 关闭 buffer |
| `:ls` | 列出 buffers |

## 常用组合

| 操作 | 快捷键 |
|------|--------|
| 删除到行尾 | `D` 或 `d$` |
| 复制整行 | `yy` |
| 删除单词 | `daw` |
| 修改单词 | `ciw` |
| 删除引号内 | `di"` |
| 选中段落 | `vip` |
| 全选 | `ggVG` |
| 全文缩进 | `gg=G` |
| 删除空行 | `:g/^$/d` |
| 行尾加分号 | `A;` |
| 多行编辑 | `Ctrl+v` → 选择 → `I` → 输入 → `Esc` |
