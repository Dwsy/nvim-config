    Neovim 基础快捷键

    模式切换

    快捷键 功能
    i 进入插入模式 (光标前)
    a 进入插入模式 (光标后)
    I 行首插入
    A 行尾插入
    o 下方新建行并插入
    O 上方新建行并插入
    v 可视模式 (字符选择)
    V 可视行模式 (整行选择)
    Ctrl + v 可视块模式 (列选择)
    Esc / Ctrl + [ 返回普通模式
    : 命令行模式
    R 替换模式

    光标移动

    基础移动

    快捷键 功能
    h 左
    j 下
    k 上
    l 右
    w 下一个单词开头
    W 下一个单词开头 (空格分隔)
    b 上一个单词开头
    B 上一个单词开头 (空格分隔)
    e 当前/下一个单词结尾
    E 当前/下一个单词结尾 (空格分隔)

    行内移动

    快捷键 功能
    0 行首
    ^ 行首非空字符
    $ 行尾
    f{char} 跳到本行下一个 {char}
    F{char} 跳到本行上一个 {char}
    t{char} 跳到 {char} 前一个字符
    T{char} 跳到 {char} 后一个字符
    ; 重复上次 f/F/t/T
    , 反向重复 f/F/t/T

    文件内移动

    快捷键 功能
    gg 文件开头
    G 文件末尾
    {n}G / :{n} 跳到第 n 行
    Ctrl + f 下翻一页
    Ctrl + b 上翻一页
    Ctrl + d 下翻半页
    Ctrl + u 上翻半页
    H 屏幕顶部
    M 屏幕中间
    L 屏幕底部
    zz 当前行居中
    zt 当前行置顶
    zb 当前行置底
    % 跳转到匹配的括号

    跳转

    快捷键 功能
    Ctrl + o 跳回上一个位置
    Ctrl + i 跳到下一个位置
    '' 跳回上次编辑位置
    '. 跳到最后修改位置
    gd 跳转到定义
    gf 打开光标下的文件

    编辑操作

    删除

    快捷键 功能
    x 删除光标字符
    X 删除光标前字符
    dd 删除整行
    D 删除到行尾
    dw 删除到下个单词
    db 删除到上个单词
    d$ 删除到行尾
    d0 删除到行首
    dgg 删除到文件开头
    dG 删除到文件末尾
    di{ 删除 {} 内内容
    di" 删除 "" 内内容
    dip 删除段落

    修改 (删除并进入插入模式)

    快捷键 功能
    cc 修改整行
    C 修改到行尾
    cw 修改到下个单词
    ci{ 修改 {} 内内容
    ci" 修改 "" 内内容
    cit 修改 tag 内内容
    s 删除字符并插入
    S 删除整行并插入

    复制粘贴

    快捷键 功能
    yy 复制整行
    Y 复制整行
    yw 复制单词
    y$ 复制到行尾
    yi{ 复制 {} 内内容
    p 粘贴到光标后
    P 粘贴到光标前
    "0p 粘贴上次复制内容
    "+y 复制到系统剪贴板
    "+p 从系统剪贴板粘贴

    撤销重做

    快捷键 功能
    u 撤销
    Ctrl + r 重做
    . 重复上次操作

    缩进

    快捷键 功能
    >> 增加缩进
    << 减少缩进
    == 自动缩进
    gg=G 全文自动缩进

    大小写

    快捷键 功能
    ~ 切换大小写
    gU{motion} 转大写
    gu{motion} 转小写
    gUU 整行转大写
    guu 整行转小写

    合并/拆分行

    快捷键 功能
    J 合并下一行
    gJ 合并下一行 (无空格)

    搜索替换

    搜索

    快捷键 功能
    /{pattern} 向下搜索
    ?{pattern} 向上搜索
    n 下一个匹配
    N 上一个匹配
    * 搜索光标下单词 (向下)
    # 搜索光标下单词 (向上)
    :noh 取消高亮

    替换

    命令 功能
    :s/old/new 替换当前行第一个
    :s/old/new/g 替换当前行所有
    :%s/old/new/g 替换全文所有
    :%s/old/new/gc 替换全文 (逐个确认)
    :5,10s/old/new/g 替换 5-10 行

    可视模式操作

    快捷键 功能
    d 删除选中
    y 复制选中
    c 修改选中
    > 增加缩进
    < 减少缩进
    u 转小写
    U 转大写
    o 跳到选区另一端
    I 块模式多行插入
    A 块模式多行追加

    文本对象

    配合 d c y v 使用:

    对象 说明
    iw 内部单词
    aw 单词 (含空格)
    is 内部句子
    as 句子 (含空格)
    ip 内部段落
    ap 段落 (含空行)
    i" "" 内部
    a" "" 包含引号
    i' '' 内部
    i) / ib () 内部
    a) / ab () 包含括号
    i] [] 内部
    i} / iB {} 内部
    it tag 内部
    at tag 包含标签

    示例: - diw - 删除单词 - ci" - 修改引号内内容 - ya} - 复制 {} 及内容 - vip - 选中段落

    标记 (Marks)

    快捷键 功能
    m{a-z} 设置标记 (文件内)
    m{A-Z} 设置全局标记
    '{mark} 跳到标记行
    `{mark} 跳到标记位置
    :marks 查看所有标记
    :delmarks a 删除标记 a

    宏录制

    快捷键 功能
    q{a-z} 开始录制宏到寄存器
    q 停止录制
    @{a-z} 执行宏
    @@ 重复上次宏
    5@a 执行宏 5 次

    窗口管理

    快捷键 功能
    :sp / Ctrl+w s 水平分屏
    :vsp / Ctrl+w v 垂直分屏
    Ctrl+w h/j/k/l 切换窗口
    Ctrl+w H/J/K/L 移动窗口
    Ctrl+w = 均分窗口
    Ctrl+w _ 最大化高度
    Ctrl+w | 最大化宽度
    Ctrl+w +/- 调整高度
    Ctrl+w >/< 调整宽度
    Ctrl+w q 关闭窗口
    Ctrl+w o 只保留当前窗口

    标签页

    快捷键 功能
    :tabnew 新建标签
    :tabc 关闭标签
    gt 下一个标签
    gT 上一个标签
    {n}gt 第 n 个标签

    文件操作

    命令 功能
    :w 保存
    :w {file} 另存为
    :q 退出
    :q! 强制退出
    :wq / :x / ZZ 保存并退出
    :e {file} 打开文件
    :e! 重新加载文件
    :bn / :bp 下/上一个 buffer
    :bd 关闭 buffer
    :ls 列出 buffers

    常用组合

    操作 快捷键
    删除到行尾 D 或 d$
    复制整行 yy
    删除单词 daw
    修改单词 ciw
    删除引号内 di"
    选中段落 vip
    全选 ggVG
    全文缩进 gg=G
    删除空行 :g/^$/d
    行尾加分号 A;
    多行编辑 Ctrl+v → 选择 → I → 输入 → Esc
